# Epic 5: Developer Experience

## Problem Statement

During Magellan pipeline runs, users lack real-time progress visibility beyond
the basic statusline. As Magellan grows, skill quality needs a validation
framework to maintain a high bar for contributions.

## Desired Outcome

Rich progress visibility during pipeline runs and a quality gate for skill
contributions.

## Features

- [F14: HUD Statusline](../features/14-hud-statusline.md)
- ~~F33: Skill Validation Framework~~ — implemented as CLAUDE.md standards

## Inspired By

- OMC: HUD with presets, agent observatory
- ECC: harness-audit for quality scoring

## Magellan Advantage

Magellan's statusline can show knowledge graph metrics (entities discovered,
contradictions surfaced, coverage percentage) alongside pipeline progress --
giving a unified view of understanding AND delivery.
